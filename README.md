# spotmaps-react-II

## Introduction

Spotmaps with React upgraded to a new workflow.

## Installation

- Install pnpm
- Clone the repo
- Rename the cloned repo
- `cd` to the folder

`pnpm install`
`pnpm run dev`

- Update the `.code.workspace`, `package.json` files to match your app namefs
- Update the `packages/client/.env` file to match your site name
